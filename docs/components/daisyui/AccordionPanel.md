# AccordionPanel

Use as part of Accordion to create a panel that can be expanded or collapsed.

## Properties

| Property | Required | Type | Description |
|----------|----------|------|-------------|
|title|No|str||
|open|No|bool||
|toggle-icon|No|str||
|name|No|str||
|Any additional properties|No|any||

## Children

Accepts any HTML content
